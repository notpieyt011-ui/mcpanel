## Installation 🚀

• sudo apt install openjdk-21-jdk -y

• curl -sL https://deb.nodesource.com/setup_23.x | sudo bash -

• apt install node.js

• apt install unzip -y

• git clone https://github.com/notpieyt011-ui/mcpanel

• unzip mcpanel.zip

• cd panel

• npm i

• npm install pm2@latest -g
  pm2 start  app.js
  pm2 startup
  pm2 save

• INSTALL CLOUDFARE AND SETUP
• HTTP localhost:3000
